# Privacy Policy - zakio Design RAG

Last updated: April 24, 2026

## Overview
This application is a personal-use tool operated by zakio.
It is used exclusively by the developer to organize design references
from their own Pinterest boards. It is not distributed to other users.

## Data Accessed
Via the Pinterest API v5, the application reads:
- The authenticated user's own boards
- Pins saved to those boards
- Basic user account information

## Data Storage
- Image URLs and pin metadata are stored on a private local server
- No data is shared with third parties
- No data is used for advertising or profiling

## Data Retention
Data is retained only for as long as the developer uses the application
for personal design reference purposes.
